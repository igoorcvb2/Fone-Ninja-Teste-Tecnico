# ERP Estoque — Teste Técnico

Pequeno ERP de estoque com cadastro de produtos, registro de compras (com cálculo
de custo médio) e vendas (com cálculo de lucro).

Stack:

- **Backend:** Laravel 13 (PHP 8.3) + MySQL 8
- **Frontend:** Vue 3 + Vite + TypeScript + Tailwind + Reka UI
- **Infra:** Docker Compose

## Como rodar

> Instruções completas no final do projeto. Por enquanto a estrutura é:

```
.
├── backend/    # API Laravel
├── frontend/   # SPA Vue
└── docker-compose.yml
```
